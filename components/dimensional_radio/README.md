# Dimensional Knob (formerly Dimensional Radio): Real-Time Music Metamorphosis for Live Performance

## Overview

Dimensional Knob (formerly Dimensional Radio) is a live performance instrument that enables DJs and performers to generate beat-synchronous variations of tracks in real-time. It's a tool for controlled musical exploration—allowing performers to navigate through coherent "dimensions" of a track while maintaining mixability and groove.

## The Problem It Solves

### For DJs and Live Performers
- **Creative Limitation**: Restricted to pre-existing recordings or pre-prepared edits
- **Preparation Overhead**: Hours spent creating edits and remixes for sets
- **Performance Risk**: Balancing spontaneity with the need for coherent mixing
- **Set Repetition**: Playing the same versions of tracks becomes predictable
- **Technical Barriers**: Remix creation requires production skills many performers lack

### For Audiences
- **Predictability**: Hearing the same versions of tracks at multiple shows
- **Limited Spontaneity**: Performances feel pre-programmed rather than reactive
- **Missed Innovation**: The boundary between DJ and live performer remains rigid

### For Music Producers
- **Unrealized Potential**: Tracks have unexplored variations that never get made
- **Remix Economics**: Remix commissions are expensive and time-consuming
- **Creative Bottleneck**: Can't explore all possible arrangements of a composition

## The Solution

Dimensional Knob is a performance instrument that generates beat-synchronous variations of loaded tracks through an intuitive knob-based interface. Performers control **how far** and **when** a track diverges from its original form—creating coherent musical journeys that preserve groove and mixability until deliberately taken elsewhere.

### Core Capabilities

**Live Performance Mode**:
- Load any track
- Turn the "dimension knob" to generate variations
- Visual feedback shows variation readiness and distance from original
- One-button return to source material
- Real-time generation maintains beat sync and tempo
- Generated variations remain mixable with other tracks

**Preparation Mode**:
- Pre-generate variation banks before performance
- Audition and select specific dimensions to load
- Create custom prompts to guide variation style
- Export favorite variations for traditional DJ use
- Build curated sets with controlled spontaneity

**Studio Mode**:
- Deeper parameter control for arrangement exploration
- Export stems and variations for production use
- Conversational prompts for specific creative directions
- Collaborative workflows with other producers
- Integration with DAWs (Ableton, FL Studio, Logic, etc.)

## How It Works

### The Technology

**Audio Analysis**:
- Real-time beat detection and tempo analysis
- Transient identification and preservation
- Harmonic and melodic structure extraction
- Genre and style classification
- Energy and intensity mapping

**Variation Generation**:
- AI-powered music transformation models
- Beat-synchronous processing ensures mixability
- Maintains core groove while varying melodic/harmonic elements
- Progressive transformation based on knob position
- Intelligent preservation of transient structure

**Performance Interface**:
- Single-knob primary control ("dimension dial")
- Visual feedback system showing variation space
- Instant return-to-source button
- Pre-generated variation bank display
- BPM and beat grid visualization

### User Experience Flow

**Quick Start (Zero Preparation)**:
1. Load track from USB, laptop, or phone
2. Play track normally
3. Turn dimension knob when ready to explore
4. System generates variation in real-time
5. Mix variation like any other track
6. Return to original or continue exploring

**Prepared Performance**:
1. Load tracks in advance
2. Pre-generate variation banks
3. Audition variations and mark favorites
4. Create setlist with planned exploration points
5. Perform with curated dimensions ready to deploy
6. Maintain spontaneity within prepared boundaries

**Studio Session**:
1. Import project stems or finished track
2. Set creative prompts and parameters
3. Generate multiple arrangement variations
4. Export preferred versions as new stems
5. Continue production with expanded options
6. Collaborate remotely using shared variation seeds
