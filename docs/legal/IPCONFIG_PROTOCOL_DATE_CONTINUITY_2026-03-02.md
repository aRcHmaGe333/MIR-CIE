# IPConfig Protocol: Disclosure Date Continuity

Protocol ID: IPCONFIG-PROT-DATE-CONTINUITY-2026-03-02

## Protocol Statement

The original disclosure date(s) in prior records still stand, even when the license or IPClaim linkage metadata is updated.

## Rule Application

1. License update dates are additive record events.
2. IPClaim linkage dates are additive record events.
3. Neither event resets or replaces original disclosure dates.
4. Any outward-facing legal/provenance summary must preserve both:
- original disclosure date(s), and
- current license/IPClaim update date(s).
