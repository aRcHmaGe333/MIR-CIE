# Exposure Plan v2 (Implemented)

## Purpose

This document operationalizes current exposure strategy for the repository with project-level independence preserved.

## Locked Strategic Posture

1. Projects are exposed individually, not as a forced integrated bundle.
2. Public now:
   - Studio Pass
   - Dimensional Knob (public name; formerly Dimensional Radio)
   - Fusion Factory
3. Deferred/private framing:
   - The Scouting Label (until wording/legal framing is hardened)
4. Cross-project mentions are conditional and only used when they strengthen ROI/risk clarity.

## Expanded Investor Definition

"Investor" includes any actor who can move execution forward:

- financial capital providers,
- authority holders,
- public/institutional access providers,
- permit/regulatory leverage actors,
- strategic operators with domain execution knowledge,
- technical ecosystem partners,
- security/compliance/logistics production enablers.

## Fusion Factory Positioning (Current)

Fusion Factory is framed as cultural infrastructure and program engine, not only as a venue startup.

First-cycle asks:
1. Venue access pathway (city/partner hall pilot access).
2. Program operations pathway (security/compliance/scheduling/neighborhood fit).
3. Technical stack pathway (audio/video/connectivity sponsorship and setup).
4. Cultural participation pathway (known + emerging + international participants).
5. Public documentation pathway (recorded outputs, workshops, artifacts).

Exposure v2 is considered implemented when:
1. Public docs reflect expanded investor definition.
2. Fusion Factory docs include operational realities (security/residential/compliance).
