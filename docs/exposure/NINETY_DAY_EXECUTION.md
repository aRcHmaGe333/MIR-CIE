# 90-Day Exposure Execution Program

## Program Scope

Public tracks:
- Studio Pass
- Dimensional Knob
- Fusion Factory

Deferred/private:
- The Scouting Label

## Days 1-14: Foundation

1. Update external-facing materials with expanded investor definition.
2. Build outreach variants:
   - tech partner packet,
   - cultural partner packet,
   - city/institution packet.
3. Build Fusion Factory pre-pilot checklist:
   - security,
   - residential impact,
   - operational command structure,
   - technical stress test requirements,
   - program schedule validation.
4. Apply provenance/IP block template to public project one-pagers.

## Days 15-45: Outbound Wave 1

1. Studio Pass outreach:
   - pilot artists,
   - strategic operators,
   - financial investors.
2. Dimensional Knob outreach:
   - performer demos,
   - ecosystem partner introductions.
3. Fusion Factory outreach:
   - city/institutional venue actors,
   - security/compliance operators,
   - technical sponsors,
   - cultural program partners.

## Days 46-75: Proof Capture

1. Execute proof activities and log evidence per project.
2. Validate Fusion Factory site-readiness criteria in pilot planning.
3. Convert qualified interest into draft LoI/MoU pathways.

## Days 76-90: Outbound Wave 2

1. Re-open target conversations using live proof evidence.
2. Prioritize projects by traction.
3. Keep Scouting Label in private exposure unless readiness gate clears.

## Evidence Log Requirements

For each conversation/pilot:
- actor type,
- project,
- ask,
- status,
- blocking risk,
- next decision date.
