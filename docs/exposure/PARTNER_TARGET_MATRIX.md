# Partner Target Matrix

## Core Music-Tech Partners

Priority partner classes and examples:
- Ableton
- Native Instruments
- Serato
- Pioneer
- Steinberg
- FL Studio ecosystem relevance (contextual influence and creator base)

## Cultural Amplification Partners

- Boiler Room
- EXIT Festival
- Comparable festival/media organizations with global reach

## Role Definitions by Project

### Studio Pass
Partner role:
- creator workflow integrations,
- artist distribution credibility,
- co-marketing with creator ecosystems.

Ask types:
- pilot artist referrals,
- ecosystem integration discussion,
- channel partnership.

### Dimensional Knob
Partner role:
- performance workflow integration,
- hardware/software compatibility narratives,
- artist demo partnerships.

Ask types:
- technical interoperability support,
- live demo environment access,
- co-branded pilot validation.

### Fusion Factory
Partner role:
- venue-grade technical sponsorship,
- broadcast/capture partnerships,
- curated event/program co-production.

Ask types:
- pilot technical stack support,
- content amplification channel,
- co-produced cultural programming.

## Outreach Segmentation Rule

Every outreach packet must identify:
1. partner category,
2. project relevance,
3. concrete ask,
4. pilot-proof benefit,
5. next-step decision point.
