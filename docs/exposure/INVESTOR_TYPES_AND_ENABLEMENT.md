# Investor Types and Enablement Model

## Definition

In this project context, an investor is any actor who can materially accelerate execution.

## Investor/Enabler Types

1. Financial Investor
- Deploys capital.
- Expects proof milestones and de-risked scaling path.

2. Strategic Operator
- Music/venue/media operator with direct execution knowledge.
- Helps with execution speed, distribution, credibility, and partnerships.

3. Institutional/Public Actor
- City, cultural authority, public venue controller, municipal partner.
- Provides access, permits, space, and civic pathway alignment.

4. Technical Ecosystem Partner
- Tool/platform firm (e.g., Ableton, Native Instruments, Serato, Pioneer, Steinberg, FL Studio ecosystem context).
- Provides integration, sponsorship, credibility, and developer channels.

5. Program Enabler
- Security, compliance, logistics, production operations.
- Makes event/program execution viable and safe.

## Application by Project

### Studio Pass
- Priority types: Financial, Strategic Operator, Technical Ecosystem Partner.
- Key enabler role: creator workflow and platform conversion acceleration.

### Dimensional Knob
- Priority types: Strategic Operator, Technical Ecosystem Partner, Financial.
- Key enabler role: live reliability validation and hardware/software integration credibility.

### Fusion Factory
- Priority types: Institutional/Public Actor, Program Enabler, Strategic Operator, Sponsor-class Financial.
- Key enabler role: venue access, compliance/safety readiness, and program operations.

### The Scouting Label (Deferred)
- Priority types: private strategic operators and legal/rights professionals.
- Public exposure intentionally restricted pending framing readiness.
